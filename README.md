# Orbit — Student Management SaaS

A production-style frontend prototype for a student-record management workspace, built with HTML5, CSS3, and vanilla ES modules.

Open [index.html](index.html) in a modern browser, or serve this directory with any static web server for the smoothest module-loading experience.

Key flows include guided manual student entry, Smart Upload → OCR processing → review → additional details → verified save, student search and filters, document review, reports, exports, settings, and a responsive workspace shell.
